# Politique de confidentialité

## Principe général

Cette application est conçue avec un principe simple: aucune collecte de données personnelles.

Aucune collecte de donnée n'est faite par l'application.

## Données personnelles

L'application ne collecte pas:
- d'identité civile,
- d'adresse postale,
- de numéro de téléphone,
- de géolocalisation,
- de profilage publicitaire.

## Finalité d'utilisation

L'utilisation de cette application est strictement réservée à la gestion du compte de l'utilisateur.

Aucune utilisation orientée exploitation commerciale des données personnelles n'est réalisée.

## Sources et vérifiabilité

Les informations utilisées pour générer les contenus proviennent de sources publiques, véridiques et vérifiables.

L'utilisateur doit conserver une démarche de vérification éditoriale avant publication.

## Partage à des tiers

L'application ne revend pas de données et ne constitue pas de base de données utilisateur à des fins de cession.

## Sécurité

Aucune base de données utilisateur dédiée à la collecte de données personnelles n'est maintenue par l'application.

## Contact

Pour toute question relative à cette politique, l'utilisateur peut se référer au responsable du projet et à la documentation du dépôt.
